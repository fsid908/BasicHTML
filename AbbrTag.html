The WHO was established in 1948.

The HTML is the standard markup language for creating web pages.

The CSS is used to style and layout web pages.

The JS is a programming language that allows you to implement complex features on web pages.

The API is a set of rules that allows different software applications to communicate with each other.

The REST is an architectural style for designing networked applications.

The GUI is a type of user interface that allows users to interact with electronic devices through graphical icons and visual indicators.

The URL is the address used to access resources on the internet.

The HTTP is the protocol used for transferring hypertext requests and information on the internet.

The SSL is a standard security technology for establishing an encrypted link between a web server and a browser.

The CMS is a software application that allows users to create, manage, and modify digital content on a website without needing specialized technical knowledge.

The SEO is the practice of optimizing a website to improve its visibility and ranking on search engine results pages.

The UX is the overall experience a user has when interacting with a product, system, or service, focusing on usability, accessibility, and satisfaction.
